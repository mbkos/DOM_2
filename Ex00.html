DOM

Métodos DOM

Os métodos permitem manipular elementos do documento HTML com JS

  1. document.getElementById("nome_do_id") - pega o elemento do HTML vai ID.
  2. innerHTML - inserir/alterar conteúdo.
  3. mudar estilo - alteração de css via js.
  4. class list - insere uma class no elemento em questão.
    • .add - adiciona uma class no elemento.
    • .remove - remove uma class no elemento.
    • .toggle - alterna entre incerir e remover a class.
  5. createElement - cria elemento HTML.
  6. remove - remove elemento.
